# Catheren Joseph — Security Engineer

I build security that enables teams to move fast — not security that slows them down.

My background is in networking, which means I understand what's actually happening on the wire before it ever reaches a policy document. That foundation, combined with hands-on cloud security engineering across AWS and Azure, shapes how I approach every problem: understand the system deeply, automate the enforcement, and make the right path the easy path.

CISSP · Azure Security Engineer · AWS Solutions Architect
