# Hey, I'm Catheren 👋

Security Engineer with a networking foundation and a cloud security focus. Security is rarely the easy path — but I try to reduce the friction enough that people don't mind taking it. That means automating the enforcement, building the right processes, and nudging teams toward secure defaults without getting in their way.

I got into security because no two days are the same. Every morning brings a different problem, a different attack surface, a different constraint to work around. That variety is what keeps me sharp — and honestly, it's just fun.

Currently exploring AI security — because the attack surface is shifting faster than most defences are.

CISSP · Azure Security Engineer · AWS Solutions Architect
